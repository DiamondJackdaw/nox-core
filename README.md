#  Nox Core
Nox Core is an XML-based metadata application profile for the archival description of video game consoles.

This project was born from the desire to develop a tailored metadata system that would allow video game console collectors to record effective, structured metadata about their collections. Incorporating semantic and syntactic elements from EAD and CDWA Lite, Nox Core seeks to marry the strengths of archival and museum metadata standards to form a lightweight, modular system that can be adapted to the needs of a given collector or repository.
