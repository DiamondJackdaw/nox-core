#  Nox Core
Nox Core is an XML-based metadata application profile for the archival description of video game consoles.

## Current version
0.9.1 (beta)

## Development history
Project development began in September 2025, with schema development beginning in November 2025.
2025.12.08: Version 0.9.0 completed (schema, element set, sample XML records)
2026.02.21: Version 0.9.1 launched (hosted on NeoCities at https://natterjack.neocities.org/noxcore)
2026.03.19: Versions 0.9.0 and 0.9.1 committed to GitHub; README file created.
